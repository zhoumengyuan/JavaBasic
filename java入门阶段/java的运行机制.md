## java的运行机制    
#### 计算机的语言一般分为编译型和解释型语言，而java既是编译型语言又是解释型语言，java分为以下几步完成编译:  
1、java利用文本编辑器编辑Java源文件（*.java）。  
2、java编译器（javac）将源文件程序编译成字节码文件（.class）文件  
3、虚拟机（解释器）解释执行  
注意：解释器是在系统平台上，Java程序没有直接和操作系统打交道，java程序在解释器里面执行，通过解释器和底层的操作系统打交道。
